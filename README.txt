🔍 Extrator de Metadados

Script simples em Python para extrair metadados e hashes (MD5/SHA256) de arquivos para análises forenses.

📚 Funcionalidades:
- Geração de hash MD5 e SHA256
- Extração de metadados de:
  - Imagens (JPEG/PNG)
  - Documentos .docx
  - Arquivos PDF

⚙️ Como usar:

1. Clone o repositório:

git clone https://github.com/seuusuario/extrator-metadados.git

2. Instale as dependências:

pip install -r requirements.txt

3. Execute o script:

python extrator.py

4. Informe o **caminho completo** do arquivo que deseja analisar.

📂 Formatos suportados:
- `.jpg`, `.jpeg`, `.png`
- `.docx`
- `.pdf`
